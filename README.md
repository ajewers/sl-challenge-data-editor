# Suade Labs JS Challenge 2019 - Alistair Jewers 
- Scenario: 1 - Data Editor
- Technology: Vue.js

## Commands
### Install Dependencies
```
npm install
```

### Compile and run with hot-reload for development
```
npm run serve
```

### Compile and minify for production
```
npm run build
```

### Lint and fix files
```
npm run lint
```

### Run unit tests
```
npm run test:unit
```
